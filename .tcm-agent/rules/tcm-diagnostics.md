# 中医诊断学规则 — 辨证要点

## 一、八纲辨证

### 表里
- 表证：发热恶寒并见、脉浮、苔薄白 — 病位在皮毛肌腠
- 里证：但热不寒或但寒不热、脉不浮 — 病位在脏腑气血
- 半表半里：寒热往来、胸胁苦满、口苦咽干、脉弦

### 寒热
- 寒证：恶寒喜暖、四肢冷、口不渴、小便清长、大便溏薄、舌淡苔白、脉迟或紧
- 热证：发热喜凉、口渴饮冷、面红目赤、小便短赤、大便干结、舌红苔黄、脉数
- 真热假寒：手足冷但胸腹灼热、脉沉但有力
- 真寒假热：身热但欲盖衣被、面赤但足冷、脉浮大但按之无力

### 虚实
- 虚证：久病、体弱、声低息微、痛处喜按、舌质嫩、脉无力
- 实证：新病、体壮、声高气粗、痛处拒按、舌质老、脉有力
- 虚中夹实：正虚为主，兼有邪实表现
- 实中夹虚：邪实为主，兼有正虚表现

### 阴阳
- 阴证：面色暗淡、身重蜷卧、静而少言、脉沉迟微弱
- 阳证：面色光亮、身轻烦躁、动而多言、脉浮数滑实
- 亡阴：汗出热黏、手足温、口渴喜冷、脉细数
- 亡阳：汗出冷清、手足厥冷、口不渴、脉微欲绝

## 二、脏腑辨证

### 心与小肠
- 心气虚：心悸气短、动则加重、自汗、面白、舌淡、脉弱
- 心阳虚：心气虚 + 畏寒肢冷、胸痛、舌淡胖、脉微
- 心血虚：心悸失眠、健忘多梦、面色萎黄、唇舌色淡、脉细
- 心阴虚：心烦失眠、五心烦热、盗汗、舌红少津、脉细数
- 心火亢盛：心烦失眠、口舌生疮、小便短赤、舌尖红、脉数
- 心血瘀阻：心悸刺痛、痛引肩背、唇舌紫暗、脉涩

### 肝与胆
- 肝气郁结：情志抑郁、胸胁胀痛、善太息、脉弦
- 肝火上炎：头胀头痛、面红目赤、口苦咽干、急躁易怒、脉弦数
- 肝阴虚：头晕目眩、两目干涩、胁痛隐隐、舌红少津、脉弦细
- 肝阳上亢：头目胀痛、面红目赤、急躁易怒、腰膝酸软、脉弦有力
- 肝风内动：眩晕欲仆、抽搐震颤、语言謇涩、脉弦

### 脾与胃
- 脾气虚：食少纳呆、腹胀便溏、神疲乏力、面色萎黄、舌淡苔白、脉缓弱
- 脾阳虚：脾气虚 + 腹痛喜温喜按、畏寒肢冷、舌淡胖、脉沉迟
- 胃阴虚：胃脘隐痛、饥不欲食、口干咽燥、舌红少苔、脉细数
- 胃火炽盛：胃脘灼痛、消谷善饥、口臭齿衄、舌红苔黄、脉滑数

### 肺与大肠
- 肺气虚：咳喘无力、气短懒言、自汗畏风、易感冒、舌淡、脉弱
- 肺阴虚：干咳少痰、痰中带血、潮热盗汗、五心烦热、舌红少苔、脉细数
- 风寒束肺：咳嗽痰白清稀、恶寒发热、鼻塞流涕、苔薄白、脉浮紧
- 风热犯肺：咳嗽痰黄黏稠、咽痛口渴、发热微恶寒、苔薄黄、脉浮数
- 痰热壅肺：咳嗽痰黄稠量多、胸闷气急、舌红苔黄腻、脉滑数

### 肾与膀胱
- 肾阳虚：腰膝酸软、畏寒肢冷、阳痿宫寒、夜尿多、舌淡胖、脉沉迟
- 肾阴虚：腰膝酸软、头晕耳鸣、五心烦热、盗汗、舌红少苔、脉细数
- 肾精不足：小儿发育迟缓、成人早衰、不孕不育、健忘痴呆
- 肾气不固：腰膝酸软、尿频遗尿、滑精早泄、胎动易滑

## 三、气血津液辨证

### 气病
- 气虚：神疲乏力、少气懒言、自汗、舌淡、脉虚
- 气滞：胀痛（走窜不定）、嗳气矢气后减、脉弦
- 气逆：咳喘（肺气上逆）、呃逆呕吐（胃气上逆）、头痛眩晕（肝气上逆）

### 血病
- 血虚：面色苍白、唇甲色淡、头晕心悸、舌淡、脉细
- 血瘀：刺痛固定不移、癥积肿块、舌紫暗有瘀斑、脉涩
- 血热：各种出血（色鲜红）、心烦、舌红绛、脉滑数
- 血寒：手足冷痛（得热减）、月经后期色暗、舌紫暗、脉沉迟

### 津液病
- 津亏：口干咽燥、皮肤干枯、尿少便干、舌红少津、脉细数
- 水肿：头面四肢浮肿、按之凹陷、小便不利
- 痰饮：咳嗽痰多、胸闷脘痞、头晕目眩、苔腻、脉滑

## 四、六经辨证要点
- 太阳病：头项强痛、恶寒发热、脉浮
- 阳明病：身热汗出、不恶寒反恶热、脉大
- 少阳病：口苦咽干目眩、寒热往来、胸胁苦满
- 太阴病：腹满而吐、食不下、自利、时腹自痛
- 少阴病：脉微细、但欲寐
- 厥阴病：消渴、气上撞心、心中疼热、饥而不欲食

## 五、卫气营血辨证要点
- 卫分：发热微恶寒、头痛咽痛、舌尖红、脉浮数
- 气分：壮热不恶寒、口渴汗出、舌红苔黄、脉洪大
- 营分：身热夜甚、心烦谵语、斑疹隐隐、舌红绛、脉细数
- 血分：高热躁狂、各种出血、斑疹显露、舌深绛

## 六、三焦辨证要点
- **上焦**（心肺）：发热微恶寒、咳嗽、头痛、舌尖红 — 温病初起
- **中焦**（脾胃）：壮热、口渴、便秘或便溏、苔黄燥 — 温病极期
- **下焦**（肝肾）：身热颧红、手足心热、口干舌燥、舌绛少苔 — 温病后期

## 七、常见证型鉴别要点

### 风寒 vs 风热
| 鉴别点 | 风寒 | 风热 |
|--------|------|------|
| 恶寒 | 恶寒重 | 恶寒轻 |
| 发热 | 发热轻 | 发热重 |
| 口渴 | 口不渴 | 口渴 |
| 咽喉 | 痒 | 痛 |
| 痰涕 | 清稀 | 黄稠 |
| 舌苔 | 薄白 | 薄黄 |
| 脉象 | 浮紧 | 浮数 |

### 气虚 vs 阳虚
| 鉴别点 | 气虚 | 阳虚 |
|--------|------|------|
| 核心表现 | 乏力、气短 | 气虚 + 寒象 |
| 寒热 | 不显 | 畏寒肢冷 |
| 舌象 | 舌淡 | 舌淡胖 |
| 脉象 | 脉弱 | 脉沉迟 |

### 血虚 vs 阴虚
| 鉴别点 | 血虚 | 阴虚 |
|--------|------|------|
| 核心表现 | 面色苍白、头晕 | 五心烦热、盗汗 |
| 寒热 | 不显 | 虚热 |
| 舌象 | 舌淡 | 舌红少苔 |
| 脉象 | 脉细 | 脉细数 |
