# 中医药安全边界规则 — 强制性约束

## 一、十八反（绝对禁忌，不可同方）

```
本草明言十八反，半蒌贝蔹及攻乌，
藻戟遂芫俱战草，诸参辛芍叛藜芦。
```

1. **乌头（川乌、草乌、附子）反**：半夏、瓜蒌（全瓜蒌、瓜蒌皮、瓜蒌仁、天花粉）、贝母（川贝母、浙贝母）、白蔹、白及
2. **甘草反**：海藻、大戟、甘遂、芫花
3. **藜芦反**：人参、西洋参、党参、丹参、玄参、沙参、苦参、细辛、芍药（白芍、赤芍）

## 二、十九畏（配伍禁忌，不可同方）

```
硫黄原是火中精，朴硝一见便相争；
水银莫与砒霜见，狼毒最怕密陀僧；
巴豆性烈最为上，偏与牵牛不顺情；
丁香莫与郁金见，牙硝难合京三棱；
川乌草乌不顺犀，人参最怕五灵脂；
官桂善能调冷气，若逢石脂便相欺。
```

1. 硫黄畏朴硝（芒硝）
2. 水银畏砒霜
3. 狼毒畏密陀僧
4. 巴豆畏牵牛（牵牛子）
5. 丁香畏郁金
6. 牙硝（芒硝）畏三棱
7. 川乌、草乌畏犀角（水牛角代）
8. 人参畏五灵脂
9. 官桂（肉桂）畏赤石脂

## 三、妊娠禁忌

### 禁用药（绝对禁用）
- **剧毒或破血逐瘀**：水蛭、虻虫、斑蝥、麝香、三棱、莪术、巴豆、甘遂、大戟、芫花、牵牛子、商陆、雄黄、轻粉

### 慎用药（谨慎使用）
- **活血化瘀**：桃仁、红花、牛膝、川芎、益母草
- **攻下通便**：大黄、芒硝、番泻叶
- **温里散寒**：附子、肉桂、干姜
- **重镇降逆**：代赭石、磁石、半夏

## 五、剂量安全边界

### 儿童剂量换算（按年龄折算）
- 新生儿：成人剂量的 1/10
- 1-3岁：成人剂量的 1/4
- 4-6岁：成人剂量的 1/3
- 7-10岁：成人剂量的 1/2
- 11-14岁：成人剂量的 2/3

### 特殊人群调量原则
- **老年人**：用量宜轻，为成人 2/3-3/4
- **体弱者**：药量宜轻，不宜峻猛
- **久病者**：缓缓图之，切忌重剂
- **过敏体质**：新药宜从小量开始

## 七、强制输出规则

**任何时候推荐药物或方剂，必须同时输出以下信息：**
1. 药物/方剂的常规剂量范围
2. 配伍禁忌（如有十八反、十九畏内容）
3. 妊娠禁忌提示（若涉及）
4. 毒性警示（若涉及毒性药物）
5. **免责声明**："以上内容仅供参考，请在执业中医师指导下使用，切勿自行用药。急重症请及时线下就医。"

## 八、禁止行为
- ❌ 不得替代医师诊断
- ❌ 不得推荐无明确出处的偏方秘方
- ❌ 不得推荐国家明令禁止的药材（如犀角、虎骨等）
- ❌ 不得对急重症（高热不退、剧烈疼痛、大出血、意识障碍等）仅给出线上建议而不建议就医
- ❌ 不得超出确定知识范围编造方剂或药物功效
