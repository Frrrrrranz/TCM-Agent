# 方剂学规则 — 方剂分类与组方原理

## 一、方剂组方结构（君臣佐使）
- **君药**：针对主病或主证起主要治疗作用的药物（方中核心）
- **臣药**：辅助君药加强疗效，或治疗兼证
- **佐药**：佐助（佐制君臣毒性烈性）、佐制（反佐引经）
- **使药**：调和诸药、引经报使

## 三、类方鉴别

### 桂枝汤类方
- 桂枝汤 → 调和营卫
- 桂枝加葛根汤 → 桂枝汤 + 葛根，治表虚项强
- 桂枝加厚朴杏子汤 → 桂枝汤 + 厚朴、杏仁，治表虚兼喘
- 小建中汤 → 桂枝汤 + 饴糖，温中补虚

### 麻黄汤类方
- 麻黄汤 → 发汗解表
- 麻黄杏仁甘草石膏汤 → 麻黄 + 石膏，辛凉宣泄
- 小青龙汤 → 麻黄 + 干姜细辛五味子，外解表寒内化水饮

### 承气汤类方
- 大承气汤 → 大黄 + 芒硝 + 厚朴 + 枳实，峻下热结
- 小承气汤 → 大黄 + 厚朴 + 枳实，轻下热结
- 调胃承气汤 → 大黄 + 芒硝 + 甘草，缓下热结

### 四君子汤类方
- 四君子汤 → 益气健脾
- 异功散 → 四君子 + 陈皮，行气化滞
- 六君子汤 → 四君子 + 陈皮 + 半夏，健脾化痰
- 香砂六君子汤 → 六君子 + 木香 + 砂仁，健脾行气

### 四物汤类方
- 四物汤 → 补血调血
- 桃红四物汤 → 四物 + 桃仁 + 红花，养血活血
- 八珍汤 → 四君子 + 四物，气血双补
- 十全大补汤 → 八珍 + 黄芪 + 肉桂，温补气血

## 四、安全提示
- 方剂中涉及毒性药物（附子、乌头、细辛、半夏等）时，必须参考 `tcm-safety.md` 中的毒性警示和剂量安全边界
- 方剂配伍必须检查十八反、十九畏（详见 `tcm-safety.md`）
- 妊娠禁忌详见 `tcm-safety.md`
- 所有方剂推荐必须附带免责声明
