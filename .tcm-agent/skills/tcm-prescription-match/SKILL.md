# Skill: 方剂匹配与推荐

## 名称
tcm-prescription-match

## 触发条件
用户给出症状或证型，要求推荐匹配的方剂；或用户询问某个方剂的组成、功效、主治、加减变化。

## 引用规则
- **方剂知识**：参考 `rules/tcm-prescriptions.md` — 君臣佐使组方结构、方剂分类速查、类方鉴别
- **中药知识**：参考 `rules/tcm-herbology.md` — 药物性味归经、功效分类、炮制与功效变化
- **安全复核**：参考 `rules/tcm-safety.md` — 十八反、十九畏、妊娠禁忌、毒性药物警示、剂量安全边界

## 工作流

### Step 1: 明确需求类型
判断用户需求属于以下哪一类：
1. **证型 → 方剂**：给定证型，推荐主方
2. **症状 → 方剂**：给定症状组合，匹配最合适的方剂
3. **方剂解析**：询问特定方剂的组成、功用、主治等
4. **方剂比较**：询问两个或多个方剂的异同
5. **方剂加减**：询问某个方剂的随症加减方案

### Step 2: 方剂匹配逻辑

**证型与方剂对应（主方推荐）**

| 证型 | 推荐主方 |
|------|----------|
| 风寒表实证 | 麻黄汤 |
| 风寒表虚证 | 桂枝汤 |
| 风热表证 | 银翘散 / 桑菊饮 |
| 少阳证 | 小柴胡汤 |
| 阳明腑实证 | 大承气汤 |
| 肺热咳喘 | 麻杏石甘汤 |
| 脾胃气虚 | 四君子汤 |
| 中气下陷 | 补中益气汤 |
| 肝郁脾虚 | 逍遥散 |
| 肝肾阴虚 | 六味地黄丸 |
| 肾阳不足 | 肾气丸 |
| 心脾两虚 | 归脾汤 |
| 血瘀胸中 | 血府逐瘀汤 |
| 痰湿内阻 | 二陈汤 / 温胆汤 |
| 阳虚水泛 | 真武汤 |
| 寒热错杂痞证 | 半夏泻心汤 |
| 肝阳上亢 | 天麻钩藤饮 |

**匹配原则（优先级递减）**
1. 主证完全匹配 → 推荐对应主方
2. 主证匹配 + 兼证差异 → 推荐主方 + 说明加减
3. 主证部分匹配 → 推荐类方中相近方剂
4. 无完全匹配 → 说明证型相近但不完全对应，推荐最接近的方剂并指出差异

### Step 3: 方剂解析（当用户查询特定方剂时）

按以下结构输出：

```
【方剂解析】
- 方名：[方剂名称]
- 出处：《[经典出处]》
- 组成：[完整药物组成及常规剂量]
- 功用：[功效总结]
- 主治：[主治病证]

【组方分析】
- 君药：[君药] — [作用说明]
- 臣药：[臣药] — [作用说明]
- 佐药：[佐药] — [作用说明]
- 使药：[使药] — [作用说明]

【临床应用】
- 辨证要点：[关键症状/舌脉]
- 随症加减：[不同兼证的加减方案]

【使用注意】
- 禁忌：[禁忌人群/证候]
- 备注：[其他注意事项]
```

### Step 4: 类方鉴别（当用户询问方剂比较时）
按以下维度进行比较：
- 相同点（组成相近/功用相似/主治相近）
- 不同点（药味差异/功用侧重/主治区别）
- 临床选择要点（什么情况下选 A vs B）

### Step 5: 安全复核
1. 检查推荐方剂中的所有药物是否符合十八反、十九畏（参考 `tcm-safety.md`）
2. 检查毒性药物剂量是否在安全范围内（参考 `tcm-safety.md`）
3. 检查妊娠禁忌（参考 `tcm-safety.md`）
4. 核实推荐方剂与用户证型是否匹配，避免误用

### Step 6: 输出建议与免责
附上免责声明，强调仅供参考，请在执业中医师指导下使用。

## 注意事项
- 方剂组成中的剂量仅供参考（因人因地因时制宜）
- 推荐方剂时优先选择经典方（经方/时方），非必要不推荐自拟方
- 如果匹配到多个候选方剂，说明各自适用场景，供用户参考
- 不可将方剂作为「成药」直接推荐给没有明确中医诊断的用户
