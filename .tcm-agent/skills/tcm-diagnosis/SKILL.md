# Skill: 中医四诊合参与辨证分析

## 名称
tcm-diagnosis

## 触发条件
用户描述症状、上传舌诊照片、或询问辨证相关问题。

## 引用规则
- **辨证依据**：参考 `rules/tcm-diagnostics.md` — 八纲辨证、脏腑辨证、气血津液辨证、六经辨证、卫气营血辨证、三焦辨证
- **安全复核**：参考 `rules/tcm-safety.md` — 十八反、十九畏、妊娠禁忌、毒性药物警示

## 工作流

### Step 1: 采集四诊信息
通过对话引导用户提供以下信息：

**望诊**
- 面色（红润/苍白/萎黄/青紫/晦暗）
- 形态（胖瘦/水肿/肌肉萎软）
- 舌象（舌质颜色、舌体形态、舌苔颜色/厚薄/润燥）
- 可引导用户描述或上传舌象照片

**闻诊**
- 声音（洪亮/低微/嘶哑）
- 气味（口臭/体气/排泄物气味）

**问诊**（核心，逐项采集）
- 主诉（主要不适症状、持续时间）
- 寒热（恶寒/发热/寒热往来）
- 汗出（自汗/盗汗/无汗）
- 头身（头痛/头晕/身痛/腰痛）
- 胸腹（胸闷/胁痛/胃脘胀满/腹痛）
- 饮食口味（纳呆/多食易饥/口渴/口苦/口淡）
- 二便（大便干结/溏薄/便秘/腹泻，小便清长/短赤/频数）
- 睡眠（失眠/多梦/嗜睡/易醒）
- 情志（急躁/抑郁/焦虑）
- 女性：月经周期/量色质/带下
- 既往史、过敏史、用药史

**切诊**
- 脉象（浮/沉/迟/数/弦/滑/细/弱/涩等）
- 如有条件可引导用户描述脉象感受

### Step 2: 综合分析辨证
基于四诊信息，按照以下步骤辨证：

1. **八纲定性**：确定表里、寒热、虚实、阴阳属性（参考 `tcm-diagnostics.md` 八纲辨证）
2. **脏腑定位**：判断病位在哪个（些）脏腑（参考 `tcm-diagnostics.md` 脏腑辨证）
3. **气血津液辨因**：分析是否存在气滞、血瘀、痰湿、水饮等病理因素（参考 `tcm-diagnostics.md` 气血津液辨证）
4. **证型判定**：综合上述分析，确定中医学证型名称（如风寒束肺证、脾胃气虚证等）

### Step 3: 输出辨证结果（按 TCM-AGENT.md 格式）

```
【辨证结果】
- 证型：[证型名称]
- 辨证依据：四诊信息与证型的对应关系分析
- 治则：[治疗法则]

【方剂建议】
- 主方：[方剂名称]（出自《经典出处》）
- 组成：[药物组成]
- 功用：[功效说明]
- 加减：[随症加减建议]

【安全提示】
- 禁忌：[禁忌人群/配伍]
- 嘱托：仅供参考，请在执业中医师指导下用药
```

### Step 4: 安全复核
1. 检查十八反、十九畏（参考 `tcm-safety.md`）
2. 检查妊娠禁忌（参考 `tcm-safety.md`）
3. 检查毒性药物剂量（参考 `tcm-safety.md`）
4. 如需急重症处理，明确建议线下就医

### Step 5: 后续建议
- 生活方式建议（饮食宜忌、起居调摄）
- 推荐进一步检查（如需要）
- 建议复诊时机

## 注意事项
- 信息不足时，主动追问缺失的关键信息
- 急重症（高热不退、剧烈疼痛、大出血等）必须建议立即线下就医
- 始终在输出末尾附带免责声明
- 可引用经典文献中的相似病证描述作为参考
