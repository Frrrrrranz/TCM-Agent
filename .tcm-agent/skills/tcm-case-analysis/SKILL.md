# Skill: 中医医案分析与学习

## 名称
tcm-case-analysis

## 触发条件
用户提供中医医案要求分析、上传病例信息要求解读、或询问经典医案中的辨证用药思路。

## 引用规则
- **辨证分析**：参考 `rules/tcm-diagnostics.md` — 八纲辨证、脏腑辨证、气血津液辨证
- **方药解析**：参考 `rules/tcm-herbology.md` — 药物性味归经、功效分类
- **方剂解析**：参考 `rules/tcm-prescriptions.md` — 君臣佐使组方结构、类方鉴别
- **安全复核**：参考 `rules/tcm-safety.md` — 十八反、十九畏、毒性药物警示

## 工作流

### Step 1: 解析医案信息
提取医案中的关键要素：

**患者基本信息**
- 年龄、性别（注意：儿、妇、老各有特点）
- 体型（肥瘦强弱）
- 体质倾向（若有明确判断）

**四诊资料**
- 望诊（神色形态、舌象）
- 闻诊（声音气味）
- 问诊（主诉、现病史、既往史、寒热汗出、饮食二便、睡眠情志等）
- 切诊（脉象、腹诊等）

**诊疗过程**
- 既往诊断（若有）
- 曾用方药（若有，记录药物及反应）
- 当前辨证、治则、方药

### Step 2: 辨证分析
基于医案信息，进行以下分析：

1. **辨证还原**：分析医案中的四诊信息与证型的对应关系（参考 `tcm-diagnostics.md`）
   - 关键症状与辨证依据的对应
   - 舌脉在辨证中的作用
2. **治则阐释**：解释治疗法则与证型的逻辑关系
   - 为什么选这个治法
   - 为什么是这个方向（如温 vs 清、补 vs 泻）
3. **方药解析**（参考 `tcm-herbology.md` 和 `tcm-prescriptions.md`）：
   - 君药的意义
   - 臣佐使的配合逻辑
   - 药物加减背后的思路
   - 剂量特点（如有特殊剂量标注说明）

### Step 3: 经典医案参照
如相关，可以引用经典医籍中的相似医案进行对照：
- 《伤寒论》《金匮要略》中相似病证的条文
- 《临证指南医案》《医宗金鉴》等后世医案
- 近现代名中医的类似验案

### Step 4: 疗效评估（如医案包含治疗结果）
1. 分析方药与疗效的因果关系
2. 评价辨证思路的合理性
3. 如有复诊调整，分析调整思路
4. 思考是否有其他可能的治疗方案

### Step 5: 输出分析结果（结构化）

```
【医案分析】
- 病证：[疾病/证型]

【辨证解析】
- 辨证依据：[四诊信息与证型的对应分析]
- 关键鉴别：[与其他相似证型的鉴别要点]

【方药详解】
- 主方：[方剂名称]
- 组方逻辑：[君臣佐使分析]
- 特色：[方药中的特别思路/用药技巧]

【学习要点】
1. [辨证方面的启示]
2. [用药方面的启示]
3. [需要特别注意的问题]

【拓展思考】
- 如遇类似病证，可参考的变通思路
```

### Step 6: 经典医案查询（当用户要求学习经典医案时）
按以下结构输出：

```
【经典医案】
- 来源：《[文献出处]》
- 医家：[医家姓名]
- 病者：[简要病情]
- 辨证：[医家辨证思路]
- 治法：[治疗法则]
- 方药：[处方用药]

【原文选录】
[引用原文段落]

【现代解读】
[从现代中医视角进行的解读分析]

【临床应用参考】
[对现代临床的启示]
```

## 注意事项
- 分析医案时保持客观，不夸大疗效
- 对于历史上著名医家的医案，尊重原文，不做过度解读
- 如果医案信息不完整，指出信息缺口及其对辨证的影响
- 不提供与现代医学病名的强行对应（如将中医证型直接等于某西医病名）
- 个人医案分析不作为诊疗建议，仅供学习参考
