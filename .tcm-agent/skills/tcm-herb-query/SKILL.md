# Skill: 中药查询与解析

## 名称
tcm-herb-query

## 触发条件
用户查询特定中药的性味归经、功效主治、用法用量、配伍宜忌等信息。

## 引用规则
- **中药知识**：参考 `rules/tcm-herbology.md` — 四气五味、归经理论、功效分类、炮制与功效变化
- **安全复核**：参考 `rules/tcm-safety.md` — 十八反、十九畏、妊娠禁忌、毒性药物警示、剂量安全边界

## 工作流

### Step 1: 识别查询目标
确定用户查询的中药名称（注意别名与正名对应）：
- 正名与别名对照（如：山萸肉 = 山茱萸、元胡 = 延胡索、二花 = 金银花）
- 炮制品区分（生/制/炒/炙/煅等）
- 确认是否正确，避免同名异物混淆

### Step 2: 信息检索
基于内置中药知识库，提取以下信息：

**基本信息**
- 药名（正名、别名）
- 性味归经（四气、五味、归经）
- 功效（核心功能描述）
- 主治（适应病证）

**临床应用**
- 常用配伍（相须相使搭配）
- 经典方剂应用（出现在哪些经典方中）
- 现代药理研究提示（需标注此为现代观点）

**用法用量**
- 常规剂量范围
- 特殊煎煮要求（先煎/后下/包煎/另煎/烊化/冲服）
- 外用用法（如适用）

**安全信息**
- 毒性等级（大毒/有毒/小毒/无毒）
- 配伍禁忌（十八反、十九畏相关内容）
- 妊娠禁忌
- 不良反应与注意事项

### Step 3: 交叉验证
1. 核对十八反、十九畏（参考 `tcm-safety.md`）
2. 确保推荐剂量在安全范围内（参考 `tcm-safety.md` 剂量安全边界）
3. 如有现代研究争议，客观标注

### Step 4: 输出结果（结构化格式）

```
【中药查询】
- 药名：[正名]（别名：[别名]）
- 性味归经：[四气五味]，归[脏腑经络]经
- 功效：[功效简述]
- 主治：[主治病证]

【临床应用】
- 常用配伍：[药物] × [药物] → [功效说明]
- 经典方剂：[方剂名称]（出自《经典》）
- 现代药理：[现代研究提示内容]（现代药理研究提示）

【用法用量】
- 内服：[剂量范围]，[煎服法说明]
- 外用：[外用方法]

【安全须知】
- 毒性：[毒性等级]
- 禁忌：[十八反/十九畏内容，妊娠禁忌]
- 注意：[特殊注意事项]
```

### Step 5: 免责声明
在输出末尾附上免责声明，强调仅供参考、请在执业中医师指导下使用。

## 注意事项
- 对于较少见或地方性用药，明确说明「此药较少在经典中出现，建议查阅专业文献」
- 国家禁止使用的药材（犀角、虎骨等）不予推荐，并解释原因
- 毒性药物必须突出警示信息
- 避免与现代西药进行直接疗效对比
